# 🎬 PREV Player - Getting Started in 5 Minutes

## Step-by-Step Setup

### Step 1️⃣: Install Node.js
Download from: https://nodejs.org/
(Choose LTS version)

### Step 2️⃣: Open Terminal/Command Prompt
```
Navigate to your Video-player folder
```

### Step 3️⃣: Install Dependencies
```bash
npm install
```
⏳ Wait 2-3 minutes...

### Step 4️⃣: Start Server
```bash
npm run dev
```

You'll see:
```
➜  Local:   http://localhost:3001/
➜  Network: http://192.168.0.111:3001/
```

### Step 5️⃣: Open in Browser
Click: http://localhost:3001

---

## 🖥️ Install on Desktop

### Windows (Chrome/Edge)
1. Open http://localhost:3001
2. Click **⬇️ icon** in address bar
3. Click **"Install"**
4. ✅ Done! Check Start Menu

### Mac (Chrome)
1. Open http://localhost:3001
2. Menu (⋮) → **"Install app"**
3. Click **"Install"**
4. ✅ Done! Check Applications

### Mac (Safari)
1. Open http://localhost:3001
2. Click **Share** → **"Add to Dock"**
3. ✅ Done! Check Dock

### Linux (Chrome/Chromium)
1. Open http://localhost:3001
2. Click **⬇️ icon** in address bar
3. Click **"Install"**
4. ✅ Done! Check App Menu

---

## 📱 Install on Phone/Tablet

### Find Your Computer's IP Address

**Windows Terminal:**
```bash
ipconfig
```
Look for "IPv4 Address" (like 192.168.0.111)

**Mac/Linux Terminal:**
```bash
ifconfig
```
Look for "inet" address

---

### Android (Chrome/Edge/Firefox)
1. On your **phone**, open browser
2. Go to: `http://192.168.0.111:3001`
3. Tap **menu** (⋮) → **"Install app"**
4. Tap **"Install"**
5. ✅ App on home screen!

### iPhone/iPad (Safari)
1. On your **iPhone/iPad**, open Safari
2. Go to: `http://192.168.0.111:3001`
3. Tap **Share** (arrow) 
4. Tap **"Add to Home Screen"**
5. Name: "PREV Player"
6. Tap **"Add"**
7. ✅ App on home screen!

## 🎮 How to Use

### Load Videos
- **Drag & Drop** video files onto player
- Or click **"Select Videos"** button
- Add multiple videos for a playlist

### Controls
| Device | Action | How |
|--------|--------|-----|
| **Desktop** | Play/Pause | Press **K** or Click video |
| **Mobile** | Play/Pause | Tap video |
| **Any** | Fullscreen | Press **F** or Double-tap |
| **Mobile** | 2x Speed | Hold screen 500ms |
| **Desktop** | Skip | Press **L** (+10s) or **J** (-10s) |
| **Any** | Time | Click time to toggle remaining |

## 🆘 Quick Troubleshooting

### "Can't connect to localhost:3001"
✅ Make sure terminal shows `npm run dev` is running
✅ Try: Ctrl+C and run again
✅ Refresh browser (Ctrl+R)

### "Install button not showing"
✅ Refresh page (Ctrl+Shift+R)
✅ Disable ad-blockers
✅ Try Chrome or Edge
✅ Clear browser cache

### "Can't connect from phone"
✅ Both on same WiFi network?
✅ Correct IP address?
✅ Server still running on PC?
✅ Check firewall settings

### "Video won't play offline"
✅ Load video online first (gets cached)
✅ Then works offline after
